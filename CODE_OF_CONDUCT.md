# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][covenant] as its
code of conduct. All contributors and participants are expected to follow it.

Please read the full text here:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be
reported to the project maintainer at **daniel.siegle@gmail.com**. All
complaints will be reviewed and investigated promptly and fairly.

[covenant]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
